## Purpose

Provide a non-developer workflow wrapper around the local `honeypot-med` CLI.

## Commands

- Setup: `python app.py start`
- Health: `python app.py doctor --json`
- Scan: `python app.py scan --input <file>`
- Protect: `python app.py protect --input <file>`
- Demo: `python app.py demo --reports-dir reports`

## Behavior

1. Prefer `scan` for first interaction.
2. Use `protect` before any release recommendation.
3. If results are confusing, point to markdown/json reports generated by demo or analyze/replay commands.
4. Do not require API keys for local mode.
