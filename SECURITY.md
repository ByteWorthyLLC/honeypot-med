# Security Policy

## Scope

`honeypot-med` is a defensive security project focused on prompt injection review, telemetry capture, and evidence exports for healthcare AI workflows.

Please report vulnerabilities that affect:

- packaged binaries and installers
- the hosted studio
- the decoy capture service
- export bundles and local file handling
- dependency or supply-chain exposure in release workflows

## Reporting

Do not open public issues for suspected vulnerabilities.

Report privately with:

- affected version or commit
- reproduction steps
- impact summary
- proof-of-concept payload if safe to share
- whether the issue affects source, packaged binaries, or Docker images

## Response Expectations

- initial acknowledgement target: 3 business days
- triage target: 7 business days
- coordinated disclosure preferred after a fix or mitigation exists

## Safe Harbor

Good-faith research that avoids privacy harm, service disruption, or unauthorized data access will be treated as responsible disclosure.
