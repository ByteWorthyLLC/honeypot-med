Hugging Face as a lab bench, not a dependency.

HF Lab generates dataset cards, system cards, artifact manifests, and a mirror plan locally. Fetching datasets is explicit, optional, and should happen only after license review.

JSONLlocal transform into packs

Generate eval cards

python app.py eval-kit --pack healthcare-challenge --outdir reports/eval-kit

Plan a mirror

python app.py hf-mirror plan --outdir reports/hf-mirror

Transform local JSONL

python app.py hf-mirror transform --input data/local-records.jsonl --outdir reports/hf-pack

HF Lab is optional and local-first by design.
