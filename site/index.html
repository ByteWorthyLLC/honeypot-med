What is Honeypot Med?
Honeypot Med is an open-source prompt-injection honeypot for healthcare AI workflows. It analyzes suspicious prompts, scores the risk, and exports a proof bundle that is easier to understand than terminal output.
Does Honeypot Med require API keys?
No. The default path is local-first. Users can run the studio and generate evidence bundles without setting up cloud billing or API keys.
Why is Honeypot Med more marketable than a typical scanner?
Because each run becomes a human-readable artifact: an HTML proof page, PDF brief, social card, JSON report, Markdown summary, and launch kit. That makes the output legible to both technical and non-technical audiences.
Who is this for?
Founders, product leaders, security teams, AI red-teamers, and healthcare builders who need a lightweight way to demonstrate prompt-injection risk in real workflows.
